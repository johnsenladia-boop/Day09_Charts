14. Which chart type did you find hardest to choose correctly? Why?  
I found the frequency distribution chart hardest to choose because it looks similar to a column chart, but it’s really about showing how often values occur. It was tricky to decide if it should be a histogram or just a regular column chart.

15. Explain in your own words the difference between WHERE and HAVING.  
WHERE filters rows before grouping, while HAVING filters groups after you use GROUP BY. So basically, WHERE works on individual rows, and HAVING works on the results of the groups.

16. When would a frequency distribution be more useful than an average?  
A frequency distribution is more useful when you want to see patterns of how often something happens, like order sizes. An average hides the details, but a frequency shows the spread and lets you see if most people order small or big amounts.
